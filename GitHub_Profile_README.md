## 🚀 Featured Projects

### 🕵️ [GNN Fraud Detection System](https://github.com/SohamJoshi07/gnn-fraud-detection)

![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?style=flat-square&logo=pytorch&logoColor=white)
![PyTorch Geometric](https://img.shields.io/badge/PyG-3C2179?style=flat-square&logo=pytorch&logoColor=white)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=flat-square&logo=streamlit&logoColor=white)
![AUC-PR](https://img.shields.io/badge/AUC--PR-0.934-28a745?style=flat-square)
![Transactions](https://img.shields.io/badge/Transactions-590K-3B7BF8?style=flat-square)

- Built on **590K IEEE-CIS transactions** using Graph Neural Networks (GCN, GraphSAGE, GAT) to detect fraud rings invisible to tabular models
- Achieved **AUC-PR 0.934** — outperforming LightGBM baseline by 10+ points on 3.5% fraud-rate data
- Integrated **GNNExplainer** for subgraph-level interpretability + live Streamlit risk scoring dashboard
